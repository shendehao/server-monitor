# 服务器管理大屏系统 — UI 设计文档

> **版本**: v1.0  
> **日期**: 2026-04-20  
> **技术栈**: Vue 3 + ECharts + Element Plus  
> **设计风格**: 深色科技风大屏

---

## 1. 设计原则

### 1.1 核心原则

- **一目了然** — 关键信息在 3 秒内可被识别
- **信息层次** — 从宏观（总览）到微观（详情）逐层深入
- **状态醒目** — 正常/警告/危险状态通过颜色立即区分
- **动态呈现** — 数据实时刷新，图表带入场和更新动画
- **大屏适配** — 主要为 1920×1080 及以上分辨率优化

### 1.2 设计规范

| 项目 | 规范 |
|------|------|
| 最小分辨率 | 1920 × 1080 (Full HD) |
| 最佳分辨率 | 2560 × 1440 (2K) / 3840 × 2160 (4K) |
| 字体 | 数字: DIN Alternate / Orbitron；中文: 思源黑体 / 系统默认 |
| 圆角 | 卡片 8px，按钮 6px，小标签 4px |
| 间距基准 | 8px 网格系统 |
| 动画时长 | 入场 600ms，更新 300ms，退场 200ms |

### 2.2 状态色定义

| 状态 | 颜色 | 色值 | 使用场景 |
|------|------|------|---------|
| 正常/在线 | 翠绿 | `#10b981` | 服务器在线、指标正常 |
| 警告 | 琥珀黄 | `#f59e0b` | CPU/内存 > 80%、轻度异常 |
| 危险 | 玫红 | `#ef4444` | CPU/内存 > 95%、服务器离线 |
| 信息 | 科技蓝 | `#3b82f6` | 一般信息、主色调 |
| 禁用/离线 | 灰色 | `#64748b` | 未启用、无数据 |

### 2.3 渐变色

| 用途 | 渐变 | CSS |
|------|------|-----|
| 顶部栏背景 | 深蓝渐变 | `linear-gradient(90deg, #0a0e27 0%, #131a35 50%, #0a0e27 100%)` |
| 卡片发光边框 | 蓝色发光 | `0 0 15px rgba(59,130,246,0.15)` |
| ECharts 面积 | 蓝色渐变 | `from: rgba(59,130,246,0.4)` → `to: rgba(59,130,246,0)` |
| 在线状态光晕 | 绿色呼吸 | `0 0 8px rgba(16,185,129,0.6)` |
| 危险状态光晕 | 红色脉冲 | `0 0 8px rgba(239,68,68,0.6)` |

## 5. 动画与交互

### 5.1 入场动画

| 元素 | 动画 | 时长 | 延迟 |
|------|------|------|------|
| 顶部栏 | 从上滑入 + 渐显 | 600ms | 0ms |
| 状态概览 | 数字翻牌 (CountUp) | 800ms | 200ms |
| 服务器卡片 | 从下滑入 + 渐显，依次延迟 | 500ms | 100ms × index |
| 趋势图 | ECharts 自带入场 | 1000ms | 600ms |
| 告警栏 | 从下滑入 | 400ms | 800ms |

### 5.2 数据更新动画

| 元素 | 动画效果 |
|------|---------|
| 数字变化 | CountUp 翻牌效果，300ms |
| 仪表盘指针 | ECharts 平滑旋转，800ms |
| 进度条 | width 平滑过渡，300ms |
| 趋势图 | 新数据点平滑追加，无闪烁 |
| 状态变更 | 颜色渐变过渡 + 闪烁提醒 |

### 5.3 交互设计

| 交互 | 效果 |
|------|------|
| 卡片悬停 | 边框发光加强，轻微上移 2px |
| 卡片点击 | 跳转服务器详情页 |
| 图表悬停 | 显示 tooltip 详细数据 |
| 告警条目点击 | 展开详情 / 跳转服务器 |
| 全屏按钮 | 进入/退出浏览器全屏 |
| 状态灯 | 在线绿色呼吸，危险红色脉冲 |

### 5.4 呼吸灯 CSS

